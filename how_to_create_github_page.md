## Commands:

1. 
   ```shell
   git checkout --orphan gh-pages
   # Creates a new branch, with no history or contents, called gh-pages and switches to the gh-pages branch
   ```

2. ```shell
   jekyll new --skip-bundle .
   # Creates a Jekyll site in the current directory
   ```
